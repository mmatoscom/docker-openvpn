# docker-openvpn

Originally forked from https://github.com/jfelten/openvpn-docker and then adjusted/upgraded.

## How to Build

```bash
	git clone https://github.com/theAkito/docker-openvpn.git
	cd docker-openvpn
	docker build .
```
